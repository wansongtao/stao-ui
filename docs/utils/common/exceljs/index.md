## 安装
::: tip 提示
使用`@stao-ui/utils`包中的`exceljs`相关方法，需要安装`exceljs`与`@stao-ui/utils`包。  
建议复制源码使用。
:::

```bash
pnpm i exceljs file-saver @stao-ui/utils
```
